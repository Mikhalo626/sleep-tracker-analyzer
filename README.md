Программа считывает файл с логами сна и рассчитывает аналитические метрики:
* Общее количество сессий сна
* Минимальная, максимальная и средняя продолжительность сессии
* Количество сессий с плохим качеством сна
* Количество бессонных ночей (отсутствие сна в интервале 00:00 - 06:00)
* Хронотип пользователя (СОВА, ЖАВОРОНОК, ГОЛУБЬ)
